# SIniestrosVialesBogotá-
Proyecto de Análisis de Datos - Siniestros viales (accidentes) ocurridos en Bogotá, desde 2015, con información relacional Actor Vial, Vehículos e Hipótesis. - Pregunta de análisis: ¿Qué factores temporales y de tipo de actor vial están más asociados a la gravedad de los siniestros viales en Bogotá?
